
#  AI Chatbot 🤖  
# Smart • Fast • Conversational

An AI-powered chatbot built using Python and Streamlit that delivers real-time, intelligent conversations with a clean and minimal interface.


#  Overview
This project simulates human-like conversations using AI. It is designed to provide a seamless chat experience with fast responses and an intuitive UI, making it perfect for learning and showcasing AI integration.


#  Features
 Real-time AI responses  
 Fast and responsive interface  
 Clean and user-friendly UI  
 Intelligent conversation handling  
 Secure API key management using `.env`  


# Tech Stack

- **Python** 🐍 
- **Streamlit**   
- **OpenAI API**
